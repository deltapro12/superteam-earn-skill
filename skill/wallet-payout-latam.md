# Wallet payout — Ecuador & LATAM

Ecuador uses **USD** as official currency — USDC/USDG → bank is straightforward.

## Receive bounty payout

1. Connect **Phantom** to Superteam Earn at signup
2. Winnings arrive as **USDC**, **USDG**, or **SOL** on Solana
3. Verify transaction on https://solscan.io

## Convert to bank dollars (Ecuador)

```
Phantom (USDC) → Binance deposit (Solana SPL) → P2P Sell → Bank transfer
```

### Steps

1. **Binance** account + KYC (cédula ecuatoriana)
2. Phantom → Send USDC → Binance (network: **Solana SPL only**)
3. P2P → **Sell** USDC → Transferencia bancaria Ecuador
4. Confirm deposit in bank app **before** releasing USDC to buyer on Binance

### Fees (typical)

| Step | Cost |
|------|------|
| Solana transfer | ~$0.01 |
| P2P spread | ~0.5–2% |
| **$75 USDC net** | ~$73–74 in bank |

## No-capital signup

- Phantom wallet: **free**, can be empty at registration
- Binance KYC: **free**, no deposit required to create account
- Capital only needed for **capital-gated** bounties (not dev/skills bounties)

## Safety

- Never share seed phrase
- Always use **Solana** network for USDC between Phantom ↔ Binance
- P2P: only release after bank credit visible
